PROJECT TITLE:Bouncing Balls Version 2
PURPOSE OF PROJECT:Very similar to version 1. Randomly generates the size and colour of n balls which will "bounce" in the window.
VERSION or DATE: August 27 2016
AUTHORS:Patricia Liu
USER INSTRUCTIONS:
Same as version 1
Characteristics of balls can be changed in the ball class.
The number of balls generated can be changed in the BouncingBalls class.

The files with the extension .class are generated when the program is compiled. They are not important.
