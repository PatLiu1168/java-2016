PROJECT TITLE:Bouncing Balls Version 1
PURPOSE OF PROJECT: Generates a randomly sized single ball which will "bounce" within the window.
DATE:August 27 2016
AUTHORS:Patricia Liu
USER INSTRUCTIONS: 
To run the program, first press on package.bluej. 
Next, right click on BouncingBallWindow and press void main(String[]args).

The speed, colour, and size of the ball can be changed in the ball class.

The files with the extension .class are generated when the program is compiled. They are not important.
Program is based off exemplar by Merill Hall.
